[TOML](https://github.com/toml-lang/toml) parser for elixir.

## Installation

The package can be installed by adding `toml_elixir` to your list of
dependencies in `mix.exs`:

```elixir
def deps do
  [{:toml_elixir, "~> 1.1.0"}]
end
```
